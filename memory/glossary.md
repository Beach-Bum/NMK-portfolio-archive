# Glossary

## Protocols & Standards
| Term | Meaning | Context |
|------|---------|---------|
| A2A | Agent2Agent Protocol | Google → Linux Foundation. Agent-to-agent communication |
| MCP | Model Context Protocol | Anthropic → Linux Foundation. Agent-to-tools |
| x402 | HTTP 402 payment protocol | Coinbase + Cloudflare. Agents pay USDC on-chain |
| ACP | Agent Communication Protocol | IBM/BeeAI. Merged into A2A early 2026 |
| DIDs | Decentralized Identifiers | W3C standard. Self-sovereign identity |
| VCs | Verifiable Credentials | Attestations tied to DIDs |
| ERC-8004 | Agent identity/reputation/validation | Ethereum standard, draft status |
| Agent Cards | A2A capability advertisements | JSON docs with capabilities, auth, endpoints |
| NIP | Nostr Implementation Possibility | Extension specs for Nostr protocol |
| AAIF | Agentic AI Foundation | Linux Foundation. Houses MCP, A2A, AGENTS.md |
